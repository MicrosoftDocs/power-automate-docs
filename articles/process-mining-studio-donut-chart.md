# Use donut charts in Process Intelligence Studio (preview)

[!INCLUDE[cc-preview-features-top-note](./includes/cc-preview-features-top-note.md)]

Donut charts in Process Intelligence Studio allow you to visualize proportional process data in a circular format with segmented categories.

[!INCLUDE[cc_preview_features_definition](includes/cc-preview-features-definition.md)]

:::image type="content" source="media/process-mining-studio-donut-chart/donut-chart-customization.png" alt-text="Screenshot of the donut chart customization panel.":::

## Label

Enter a custom label for the chart.

## Data

- **Each part represents**: From the dropdown menu, select what each part represents from the dropdown (for example, **Edges**, **Variants**, or **Attribute**).

- **Attribute name**: Select the specific attribute for the category.

## Metric type

Choose the metric type (for example, **Frequency**, **Performance**, **Rework**, or **Finance**).

## Size of each part shows

Select the metric that determines segment size.

## Customizations

**Limit records to show**: Set the number of records to display. The default is 5.

> [!NOTE]
> You can show a maximum of 10 in the donut charts.
