# Donut chart

![Donut chart customization panel](media/process-mining-studio/donut-chart-cuztomization.png)

## Label

Enter a custom label for the chart.

## Data

**Each part represents**
Select what each part represents from the dropdown (e.g., Edges).

**Attribute name**
Select the attribute type to display.

**Metric type**
Choose the metric type (e.g., Frequency).

**Size of each part shows**
Select the metric that determines segment size.

## Customizations

**Limit records to show**
Set the number of records to display (default: 5).

Note: Maximum of 10 can be shown in the donut charts.
