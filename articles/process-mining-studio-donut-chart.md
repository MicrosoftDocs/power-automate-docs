# Donut chart

:::image type="content" source="media/process-mining-studio/donut-chart-cuztomization.png" alt-text="Screenshot of the donut chart customization panel.":::

## Label

Enter a custom label for the chart.

## Data

**Each part represents**
Select what each part represents from the dropdown (e.g., Edges, Variants, Attribute).

**Attribute name**
Select the specific attribute for the category.

**Metric type**
Choose the metric type (e.g., Frequency, Performance, Rework, Finance).

**Size of each part shows**
Select the metric that determines segment size.

## Customizations

**Limit records to show**
Set the number of records to display (default: 5).

> [!NOTE]
> Maximum of 10 can be shown in the donut charts.
